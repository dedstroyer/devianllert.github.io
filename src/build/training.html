Vanilla JS

? Vanilla JS означает, что используется нативный JavaScript, т.е. без всяких модификаций, фреймворков и т.д.

Эту функцию легко сделать, для начала нужно взять значения из каждого input делается это с помощью document.querySelector(".").value.
